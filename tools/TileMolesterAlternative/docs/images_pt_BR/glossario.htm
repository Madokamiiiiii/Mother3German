Glossário

 
Arquivo batch
batch file = arquivo em lote = arquivo armazenado, contendo uma seqüência de comandos do sistema, que quando ativado executa todos esses comandos automaticamente, usado em vez de ter que digitar os comandos um a um;

Binário
Literalmente, significa que um arquivo está no formato composto por um conjunto de uns e zeros; na prática, siginifica que um arquivo foi formatado de modo a ser legível somente por alguns aplicativos, ou que é um arquivo executável.

Bit
bit; (a) = BINARY DIGIT = DÍGITO BINÁRIO menor unidade na notação numérica binária, que pode ter o valor 0 ou 1; (b) menor unidade de dado que um sistema pode tratar;

Bitmap
mapa de bits; Uma representação bidimensional de dígitos binários (bits) para uso em representação um a um com um dispositivo de apresentação para produzir imagrns gráficas.

Bit mask
mask bit = bit de máscara = um bit usado para selecionar o bit desejado de uma palavra ou cadeia

Bugs
erro, falha; 1 (a) erro num programa de computador que o faz executar incorretamente

Byte
grupo de bits ou dígitos binários (geralmente oito) que o computador opera como uma unidade simples;

Classe
Termo da programação orientada a objetos, classe refere-se a um método de definição de um conjunto de objetos relacionados, capaz de herdar ou compartilhar determinadas características.

Checksum
or check total soma de verificação ou total de verificação; programa que verifica se o dado está correto, através da sua soma e comparação com um valor armazenado;

Codec
= CODER/DECODER = CODIFICADOR/DECODIFICADOR

Compressão de dados
data compression = meios de reduzir o tamanho dos blocos de dados através da remoção de espaços, seções vazias e material não usado.

Emulador
emulator; software ou hardware que permite que uma máquina (ou sistema) se comporte igual a outra.

Entidade
entity; assunto sobre o qual o dado armazenado em um arquivo ou base de dados se refere.

Estrutura de diretório
Organização hierárquica de arquivos em uma árvore de diretórios.

Front-end
Lado da intyerface do usuário de um aplicativo, o front-end é o que os usuários vêem e com o qual interagem

GUI
= GRAPHICAL USER INTERFACE = INTERFACE GRÁFICA DO USUÁRIO interface com o usuário que se utiliza de janelas e ícones que, manipulados pelo mouse, facilitam o trabalho de operação de um software. O programa Windows da Microsoft é um exemplo de GUI.

  

Header
Cabeçalho ou registro inicial; (a) informação no início de uma lista de dados referente ao resto dos dados;

ISO
= INTERNATIONAL STANDARDS ORGANIZATION = ORGANIZAÇÃO INTERNACIONAL DE PADRÕES; ISO/OSI

Java
Linguagem de programação de alto nível, desenvolvida para escrever programas de sistemas estruturados, ou orientado a objeto, leve, independente de arquitetura e portável para qualquer plataforma que suporte o interpretador.

Java Runtime Environment
Ambiente de Execução Java, é um conjunto de biblioteca de classes, máquina virtual e outros arquivos necessário para interpretar programas escritos em java.

Matriz
(a) arranjo de números ou itens de dados organizados em linhas e colunas;

Offset
offset value or offset word = valor ou endereço relativo = valor a ser adicionado a um endereço base para produzir um endereço final indexado.

Paleta
palette; variação de cores que podem ser usadas (em uma impressora ou monitor de computador).

Pixel
or picture element pixel ou elemento de imagem; menor unidade ou ponto de um monitor de vídeo cuja cor ou brilho pode ser controlado.

Raw
bruto; no estado original ou não processado; ± raw data = dados brutos = (i) informação que ainda não foi introduzida em um sistema de computação;

RGB
= RED, GREEN, BLUE = VERMELHO, VERDE, AZUL sistema monitor de alta definição que usa três sinais de entrada separados que controlam os feixes de imagens nas cores vermelha, verde e azul.

Runtime
or run-duration tempo de execução ou duração da execução; 1 (a) intervalo de tempo que um programa leva para executar; (b) tempo durante o qual um computador está executando um programa; 2 (operação) executada apenas quando um programa está sendo executado;

String
cadeia ou seqüência; quaisquer séries de caracteres alfanuméricos ou palavras consecutivas que são manipuladas e tratadas como uma unidade pelo computador;

Tag
identificador; (a) parte de uma instrução de computador; (b) caracteres de identificação liga-dos a um arquivo ou item (de dados)
